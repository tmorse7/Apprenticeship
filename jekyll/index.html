About Jekyll

Jekyll is static website framework that can be downloaded as a ruby gem to make webpages such as this one! It is able to create a webpage through the use of command line combining file types of many sorts such as html, css, Markdown, and Javascript (there's probably more). I've been using a local host preview to edit this site.

Some commands to remember are:

  • Installing Jekyll (linux/MAC): gem jekyll install
  • Create Jekyll Source Folder: jekyll build
  • Launch Jekyll Site: jekyll serve (--watch)*

* --watch used to update on refresh, which is optional.
