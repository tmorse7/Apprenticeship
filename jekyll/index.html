About Jekyll

Jekyll is a ruby gem that is used to make static websites, such as this one! It is able to create a webpage through the use of command line. I've been using a local host preview to edit this site.

Some commands to remember are:

  • Installing Jekyll (linux/MAC): gem jekyll install
  • Create Jekyll Source Folder: jekyll build
  • Launch Jekyll Site: jekyll serve (--watch)*

* --watch used to update on refresh, which is optional.
