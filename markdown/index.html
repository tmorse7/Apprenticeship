About Markdown

Markdown is a programming language that lets you make simple HTML based changes to text documents which use the file type of .md. To me, it feels almost like a shorthand version of HTML formatting.

Some notable syntax that Markdown uses are:

  • Headers: h1, h2, h3, h4, h5, h6
  • Bold: **(text)** or __(text)__
  • Italic: *(text)* or _(text)_
  • Ordered List: 1. (text) (ENTER)
  • Unordered List: * or - (text) (ENTER)
  • Links: [text](link)
  • Images: ![alt text*](link)
  • Referencing: [reference variable] and then later defined: [ref var]: reference item

* Alt text is the text that appears when the image isn't loaded.

One big thing to note is that if you double space after something, it will create a new line. This comes into use with this like lists.
